>>> Instruction for compiling:

''' bash
cd ./src
javac Client.java
javac Server.java
'''
The files could be compiled with JDK 11 environment.

>>> Instruction for testing:

''' bash
cd ./src
java Client {PERSONALIZED PORT}
java Server {YOUR SERVER HOSTNAME} {PORT}
'''
Server hostname is either local host name (should you run both files on local machine) or ip-address if testing over web.
